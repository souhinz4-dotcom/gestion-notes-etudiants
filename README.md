# Projet Java — Gestion de notes des étudiants

## Membres du groupe
- Zainabou
- Mamadou

## Description
Programme Java qui lit les notes des étudiants depuis un fichier CSV,
calcule leurs moyennes, les classe par ordre décroissant,
et sauvegarde les résultats dans un fichier texte.

## Fonctionnalités
- Lecture des données depuis un fichier CSV (etudiants.csv)
- Calcul automatique de la moyenne de chaque étudiant
- Classement des étudiants par moyenne (du plus haut au plus bas)
- Sauvegarde des résultats dans un fichier texte (resultats.txt)
- Affichage des résultats dans la console

## Comment exécuter le programme

### Étape 1 — Compiler
javac *.java

### Étape 2 — Exécuter
java Main

### Résultat attendu
Le programme affiche le classement dans la console
et crée un fichier resultats.txt automatiquement.
